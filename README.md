# Naturals Training Academy - Landing Page

A premium, modern, and highly professional landing page designed for the Naturals Training Academy in Pollachi. This frontend application is built to provide an editorial, high-end user experience to showcase beauty and wellness courses.

## 🚀 Tech Stack
- **React 19**
- **Vite** (Build Tool)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## ✨ Features
- **Modern UI/UX:** Clean, editorial design with a White + Purple (#6D28D9) color scheme.
- **Smooth Animations:** Integrated scroll reveals, stagger effects, and floating glassmorphism components via Framer Motion.
- **Fully Responsive:** Carefully optimized for mobile, tablet, and desktop viewports to ensure zero horizontal scrolling or cut-offs.
- **SEO Ready:** Semantically structured with proper tags for optimal search engine indexing.

## 📦 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd naturals
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🌐 Deployment (Vercel)
This project is configured and ready for 1-click deployment on [Vercel](https://vercel.com).
The `vercel.json` file is included in the root directory to handle client-side routing automatically.

1. Push your code to a GitHub repository.
2. Import the project in your Vercel Dashboard.
3. Vercel will automatically detect Vite and configure the build settings.
4. Deploy!
